Що ж заважає писати чистий код?

  • Ігнорування
  • Впертість

  • Зарозумілість

Don't Repeat Yourself

Найменування

  • Для змінних, властивостей та параметрів використовувати іменники
    indexer, currentUser, PriceFilter
  • Для методів та функцій — дієслова
    SaveOrder(), getDiscounts(), RunPayroll()
  • Легковимовні та недвозначні
    recdptrl = received_patrol['record_department_role']
  • Стандарт іменування
    camelCase
    PascalCase
    snake_case
    Неважливо! Просто майте стандарт та будьте послідовними

Коментарі

  • Вони неправдиві
  • Вони не роблять поганий код хорошим

Проте коментарі бувають "хороші"

  • Для опису чи роз'яснення
    // Sample output: Oct 20, 2016 - 12:34:15
  • Для попередження про неочевидні наслідки
    // We do a deep copy of this collection to make sure we don't change it
  • Можуть бути використані для TODO
    Особливо коли IDE їх підтримує
    Повинні бути тимчасові
