Flexbox

Flexbox Layout (Flexible Box) призначено для більш ефективної роботи із вирівнювання, розміщення та розподілу простору між елементами в контейнері, навіть якщо їх розмір невідомий або динамічний

Головна ідея — надати контейнеру можливість змінювати свою висоту, ширину (і порядок) аби якнайкраще заповнити доступне місце. Гнучкі контейнери розширюють свої елементи щоб заповнити вільне місце, або стискають, щоб запобігти переповненню.

  • row-reverse зправа наліво
  • column згори донизу
  • column-reverse знизу догори

justify-content

  • flex-start (за замовчуванням) елементи розміщені на початку лінії

  • center елементи розміщені по центру

  • space-around елементи рівномірно розподілені по лінії із однаковими відступами

Властивості елементу

order

flex-grow

.item {
  flex-grow: <number>
} За замовчуванням 0
Не може бути від'ємним

flex-shrink

.item {
  flex-shrink: <number>
} За замовчуванням 1
Не може бути від'ємним

flex-basis

.item {
  flex-basis: <length> | auto;
} За замовчуванням auto
Якщо стоїть 0, то flex-basis не враховується

flex

Скорочення для flex-grow, flex-shrink і flex-basis.

За замовчуванням 0 1 auto

align-self

  • auto (за замовчуванням, якщо є контейнер) наслідувати контейнер
  • stretch (за замовчуванням, якщо немає контейнеру) елементи заповнюють контейнер
  • flex-start елементи зміщені до початку
  • flex-end елементи зміщені до кінця
  • center елементи розміщені по центру
  • baseline елементи вирівняні відповідно до їх базису
