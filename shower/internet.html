Internet

Що відбувається від набору URL у браузері до відображення веб-сторінки?

Long story short

Отримання IP-адреси

Комп'ютер надсилає запит до серверу DNS (Domain Name System), щоб знайти який саме IP пов'язаний із доменом, що ви ввели. Якщо ж відразу ввести IP, то ви відразу потрапите на сервер без DNS.

Надсилання інформації на сервер вебсайту

Комп'ютер надсилає HTTP GET запит на сервер, IP якого отримав.

Отримання інформації від серверу

Сервер опрацьовує ваш запит і повертає назад певні дані. Для веб-сторінки це html, css тощо.

Користувач бачить сайт

Браузер опрацьовує повернений результат і відображає його користувачеві.
