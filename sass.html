SASS

Syntactically Awesome Style Sheets

Встановлення і використання

http://sass-lang.com/install
  • Встановити ruby (http://rubyinstaller.org/)
  • Встановити Sass `gem install sass` або `sudo gem install sass`

Відслідковувати каталог

`sass --watch path/sass-directory`

Генерувати файли у визначеній папці

`sass --watch path/sass-directory:path/css-directory`

Відслідковувати окремий файл

`sass --watch path/sass-directory/styles.scss` 
                
/* SASS */
.content
    border: 1px solid red
    color: black
                
                
/* SCSS */
.content {
    border: 1px solid red;
    color: black;
}

.border {
    padding: 10px;
    margin: 10px;
}

Partials

Можна створювати часткові файли стилю, які містять маленькі частини CSS, і які потім можна включати до інших sass-файлів.

Імпортування

                
// _reset.scss
* {
  margin: 0;
  padding: 0;
}

// base.scss
@import 'reset';

body {
  font: 100% Helvetica, sans-serif;
  background-color: #efefef;
}

  padding: 0;

Extend/Inheritance

                
message {
  border: 1px solid #ccc;
  padding: 10px;
  color: #333;
}

.success {
  @extend .message;
  border-color: green;
}

.error {
  @extend .message;
  border-color: red;
}
                
                
.message, .success, .error, .warning {
  border: 1px solid #cccccc;
  padding: 10px;
  color: #333;
}

.success {
  border-color: green;
}

.error {
  border-color: red;
}

@mixin with arguments

    style: dashed;

@function

                
$grid-width: 40px;
$gutter-width: 10px;

@function grid-width($n) {
  @return $n * $grid-width + ($n - 1) * $gutter-width;
}

#sidebar { width: grid-width(5); }
